# Taller de Data Science 📊

Este repositorio contiene los ejercicios, consultas y el desarrollo práctico del taller correspondiente al **Diplomado en Data Science**. El proyecto se enfoca en la creación, gestión y consulta de bases de datos relacionales utilizando SQL, así como en el análisis de datos mediante Python.

---

## 🛠️ Tecnologías y Herramientas

* **SQL**: Creación de bases de datos, consultas estructuradas y análisis relacional.
* **Python**: Procesamiento de datos y desarrollo en Jupyter Notebooks (`.ipynb`).
* **Pandas**: Manipulación y análisis del conjunto de datos de consumo de combustible.
* **Markdown**: Documentación del proceso y resultados visuales.

---
